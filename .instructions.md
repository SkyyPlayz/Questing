# Detail Design and Editing Instructions

## Scope
- These instructions apply to all files and technologies in this workspace.
- The agent will be used primarily for detailed design work and editing tasks.

## User Preferences
- The user is unsure which plugins or tools may be needed for tasks.
- If the agent requires a plugin, tool, or extension that is not already present, it must ask for permission before proceeding.
- When requesting permission, the agent must explain why the tool or plugin is needed and how it will help with the current task.

## Enforcement
- Always pause and request explicit user approval before installing or using new tools, plugins, or extensions.
- Provide a brief explanation of the benefit or necessity of the requested tool.

## Example Prompts
- "I need to install the Prisma extension to help with database schema editing. May I proceed? This will enable advanced schema validation."
- "To generate a detailed UI preview, I recommend installing the VS Code Live Server extension. Would you like to continue? This will allow you to see real-time changes."
